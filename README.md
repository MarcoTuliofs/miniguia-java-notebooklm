# miniguia-java-notebooklm
Criar um professor e auxiliador que sirva como um guia educacional abrangente para dominar a linguagem de programação Java . Para auxiliar com um caminho estruturado do básico ao avançado.
🎯OBJETIVO:
Criar um professor e auxiliador que sirva como um guia educacional abrangente para dominar a linguagem de programação Java . Para auxiliar com um caminho estruturado do básico ao avançado.

🔗FONTES DE TEXTO:  
https://dev.java/learn/  
https://java-programming.mooc.fi/  
https://homepages.dcc.ufmg.br/~bigonha/Livros/java.pdf

Perguntas Estratégicas Elaboradas ❔  

PERGUNTA 1: Explique o conceito de Programação Orientada a Objetos em Java para um iniciante ?

RESP: O orientador respondeu à pergunta de forma clara e didática, explicando os principais conceitos da Programação Orientada a Objetos (POO) em Java, durante a resposta foram abordados temas como classes e objetos, abstração, encapsulamento e a proteção de dados, herança e polimorfismo. Além disso, foram apresentados elementos importantes da linguagem Java, como métodos, construtores e interfaces, acompanhados de exemplos para melhor compreenção.

VARIAÇÃO DE PROMPT:
PERGUNTA 1: Programação Orientada a Objetos ?

RESP: Nesta pergunta, a resposta apresentada foi mais simples e genérica, abordando apenas os conceitos básicos de POO. Apesar de estar correta, a explicação não trouxe muitos detalhes ou exemplos práticos ou aprofundamento dos conceitos, o que trouxe uma certa limitação a compreensão do conceito.  

PERGUNTA 2: O que são Collections em Java e quais as diferenças entre ArrayList, LinkedList e HashMap ?  

RESP: A resposta foi bem elaborada e didática, iniciando com uma explicação geral sobre o conceito de Collections em Java. Em seguida, foram apresentados os conceitos de ArrayList, LinkedList e HashMap de forma detalhada, explicando suas características e funcionamento. Além disso, o orientado realizou uma comparação entre as três estruturas, destacando aspectos como o tipo de estrutura utilizada, a forma de armazenamento dos dados, a eficiência no acesso e manipulação dos elementos e os cenários mais adequados para utilização de cada uma. Essa abordagem facilitou a compreensão das diferenças entre as estruturas e auxiliou na identificação da melhor opção para cada situação de desenvolvimento.
