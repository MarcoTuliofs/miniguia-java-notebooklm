Programação Orientada a Objetos (POO)

A Programação Orientada a Objetos é um paradigma de programação baseado na criação de objetos que representam elementos do mundo real.

PRINCIPAIS CONCEITOS:  

Classe: Modelo que define características e comportamentos de um objeto.

Objeto: Instância de uma classe.

Encapsulamento: Protege os dados internos de um objeto por meio de modificadores de acesso e métodos.

Herança: Permite que uma classe herde características e comportamentos de outra.

Polimorfismo: Possibilita que um mesmo método tenha comportamentos diferentes dependendo do contexto.

Abstração: Representa apenas as características essenciais de um objeto.

Interfaces: Definem contratos que as classes devem implementar.

Collections em Java: O Framework Collections fornece estruturas de dados prontas para armazenamento e manipulação de informações.

ArrayList
- Baseado em array dinâmico.
- Acesso rápido aos elementos.
- Bom para consultas frequentes.
- Inserções e remoções no meio da lista podem ser mais lentas.
   
LinkedList
- Baseado em lista encadeada.
-Inserções e remoções são mais eficientes.
- Acesso aos elementos é mais lento.
  
HashMap
- Armazena dados em pares chave-valor.
- Busca extremamente rápida.
- Não mantém ordem dos elementos.

GLOSSÁRIO

Java

Linguagem de programação orientada a objetos desenvolvida pela Sun Microsystems.

POO (Programação Orientada a Objetos)

Paradigma baseado na criação e interação entre objetos.

Classe: Estrutura que define atributos e métodos.

Objeto: Instância criada a partir de uma classe.

Método : Função pertencente a uma classe.

Construtor: Método especial utilizado para inicializar objetos.

Encapsulamento: Proteção dos dados internos de uma classe.

Herança: Mecanismo de reutilização de código entre classes.

Polimorfismo: Capacidade de um método assumir comportamentos diferentes.

Interface: Contrato que define métodos obrigatórios para uma classe.

Collection: Estrutura de dados utilizada para armazenar conjuntos de objetos.

ArrayList: Lista dinâmica baseada em arrays.

LinkedList: Lista dinâmica baseada em nós encadeados.

HashMap: Estrutura que armazena pares chave-valor.

Debug: Processo de identificação e correção de erros em um programa.

Prompt: Instrução ou pergunta enviada para a Inteligência Artificial.

Aprender um novo conceito

Aprofundar conhecimentos

Explicação de código
